Prontos Pro Mundo — Intercâmbio Público Educação

O projeto Prontos Pro Mundo é uma iniciativa da Secretaria da Educação do Estado de São Paulo que visa proporcionar aos estudantes da rede pública estadual uma experiência internacional transformadora. Lançado em 2024, o programa é o maior intercâmbio público estudantil já realizado no Brasil. Ele oferece aos alunos a oportunidade de estudar em países de língua inglesa como Canadá, Reino Unido, Austrália, Nova Zelândia e Irlanda, com todos os custos cobertos pelo governo estadual.

A seleção dos participantes começa no Ensino Fundamental, através do desempenho na prova do SARESP. Os 70 mil alunos com as melhores notas são convidados a participar de um curso intensivo de inglês online pela plataforma SPEAK. Após essa formação, os estudantes passam por uma prova presencial de proficiência. Quem atinge os critérios exigidos avança para a etapa final do processo seletivo.

Os aprovados recebem suporte completo: passagens aéreas, hospedagem, alimentação, aulas no exterior, passaporte, visto e até bolsa para despesas pessoais. A experiência é planejada para durar até um semestre letivo, com embarques previstos para os dois semestres de 2025.

Mais do que um intercâmbio, o Prontos Pro Mundo representa uma mudança significativa na educação pública paulista. Com investimento de cerca de R$ 120 milhões, o programa valoriza o mérito acadêmico e promove inclusão, equidade e desenvolvimento de competências para o século XXI.

Arthur Berlofa: Meu Caminho até o Prontos Pro Mundo

Este projeto apresenta minha trajetória pessoal até ser selecionado para o intercâmbio internacional do programa Prontos Pro Mundo, promovido pela Secretaria da Educação de São Paulo.

Meu nome é Arthur Anttoni Tanaka Berlofa, sou estudante do ensino médio e, com dedicação ao estudo de inglês e programação, fui aprovado para representar minha escola e o Brasil em uma experiência de intercâmbio educacional no exterior.

Participei das etapas do processo seletivo: destaquei-me no SARESP, fui aprovado no curso online da plataforma SPEAK e alcancei a pontuação necessária na prova de proficiência. Minha jornada foi registrada em uma matéria publicada no portal educacional da região.

Explorando o Universo com HTML e JavaScript

Este projeto é uma demonstração prática das minhas habilidades com HTML e JavaScript. Criei uma experiência interativa inspirada no universo da astronomia, onde as pessoas usuárias podem aprender sobre constelações, eclipses e muito mais.

Utilizei HTML para estruturar o conteúdo e JavaScript para criar interações, como transições de tela, mudanças de cenário e respostas visuais a cliques do usuário. O visual é didático, acessível e envolvente.

Entre os destaques técnicos, estão:

  • Manipulação do DOM com JavaScript puro
  • Uso de eventos como onclick e addEventListener
  • Design responsivo com CSS
  • Apresentação interativa com efeitos visuais

Esse projeto me ajudou a consolidar meus conhecimentos em programação web e serviu como base para outros trabalhos mais avançados.
